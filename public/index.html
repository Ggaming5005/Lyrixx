lyrixx API

A Node.js API that extracts synced lyrics from Musixmatch via Spotify.

Check the source code for details. It's well-commented! Feel free to ask any questions.

Endpoints

üîç Search lyrics by Track ID

GET https://lyrixxx.vercel.app/getLyrics/{trackId}

Retrieve lyrics using the Spotify Track ID.

Example Request: 
GET https://lyrixxx.vercel.app/getLyrics/123456789
Example Response: 
{
  "trackId": "123456789",
  "title": "Lucid Dreams",
  "artist": "Juice WRLD",
  "lyrics": "I still see your shadows in my room..."
}

üîç Search lyrics by Artist & Track Name

GET https://lyrixxx.vercel.app/getLyricsByName/{artistName}/{trackName}?remix={true/false}

Retrieve lyrics using the artist and track name.

Optional: Add ?remix=true to get only official remix versions.

Example Request (Remix): 
GET https://lyrixxx.vercel.app/getLyricsByName/JuiceWRLD/LucidDreams?remix=true
Example Request (Original): 
GET https://lyrixxx.vercel.app/getLyricsByName/JuiceWRLD/LucidDreams
Example Response: 
{
  "trackName": "Lucid Dreams",
  "artist": "Juice WRLD",
  "isRemix": false,
  "lyrics": "I still see your shadows in my room..."
}

API Usage

  1. Get Lyrics by Track ID:
    • Make a GET request to https://lyrixxx.vercel.app/getLyrics/{trackId}.
    • Replace {trackId} with the actual Spotify Track ID.
  2. Get Lyrics by Artist & Track Name:
    • Make a GET request to https://lyrixxx.vercel.app/getLyricsByName/{artistName}/{trackName}?remix={true/false}.
    • Replace {artistName} and {trackName} accordingly.
    • Use ?remix=true if looking for remix versions.
